Welcome to collegebaseball's documentation!
===========================================

**collegebaseball** is an open-source college baseball analysis package for Python.
It includes functionality for data acquisition and calculation of advanced metrics,
including an intuitive API for *stats.ncaa.org*

.. note::

   This project is under active development.
